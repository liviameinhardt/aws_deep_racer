Relatrio de Treinamento: Funes de Recompensa para o AWS DeepRacer

Introduo

Para os primeiros testes de treinamento do carro, buscamos inspirao em artigos publicados por participantes de destaque no ranking de corridas do AWS DeepRacer Virtual Circuit. Essa abordagem visava otimizar o desenvolvimento de funes de recompensa eficazes, considerando as limitaes de crditos para treinamento.

Selecionamos trs publicaes de blog que apresentavam implementaes simples, porm eficazes, e cujas estratgias se destacaram no desempenho competitivo:

  1. "Craft a Powerful Reward Function for AWS DeepRacer Student League"
  2. "How I Got Into The Top 2% In AWS DeepRacer"
  3. "How We Broke Into The Top 1% of the AWS DeepRacer Virtual Circuit"

Estratgias de Recompensa

As ideias exploradas nos artigos compartilhavam algumas similaridades, mas tambm apresentavam distines importantes:

  1. Primeiro e segundo artigos:

    • Ambas as abordagens focavam em recompensar o veculo com base na proximidade do centro da pista, penalizando desvios excessivos.
    • O segundo artigo, entretanto, adicionava um controle mais detalhado, relacionando a distncia ao centro com a rotao das rodas do carro, conferindo maior preciso.

  2. Terceiro artigo:

    • A principal diferena era o uso de informaes de estado anteriores na funo de recompensa, como velocidade e ngulo de direo. Essa abordagem introduzia uma perspectiva de longo prazo, buscando suavidade e consistncia na direo.

Implementao e Treinamento

Com base nas ideias e cdigos sugeridos pelos blogs, desenvolvemos duas verses de funes de recompensa que combinavam os conceitos mais robustos dos artigos analisados. Os treinamentos foram realizados utilizando hiperparmetros padro durante 1 hora na pista [NOME DA PISTA].

Primeira Funo de Recompensa

A primeira funo de recompensa foi projetada para promover uma direo suave, rpida e eficiente, com foco em trs aspectos principais: aderncia linha central, controle de velocidade e eficincia no progresso. A funo utiliza os seguintes critrios para calcular a recompensa:

  1. Adeso Linha Central:

    • O veculo recompensado de acordo com sua proximidade ao centro da pista.
    • A recompensa proporcional distncia, com uma curva que penaliza desvios de forma progressiva.

  2. Controle de Velocidade:

    • Velocidades ideais so definidas de acordo com a inclinao do ngulo de direo:
      • Para retas (ngulo de direo < 10): a velocidade alvo 2.5 m/s.
      • Para curvas (ngulo de direo 10): a velocidade alvo 1.8 m/s.
    • Penalidades so aplicadas para discrepncias significativas entre a velocidade atual e a ideal.

  3. Penalizao para Curvas Fechadas com Alta Velocidade:

    • Um fator de penalizao aplicado se o ngulo de direo exceder 15 enquanto o veculo estiver a velocidades superiores a 1.8 m/s, reduzindo a recompensa final.

  4. Eficincia no Progresso:

    • A funo considera o progresso percentual alcanado em relao ao nmero de passos dados. Isso incentiva trajetrias eficientes, promovendo uma combinao de velocidade e preciso.

  5. Combinao Ponderada:

    • Os fatores so combinados com diferentes pesos para equilibrar os critrios:
      • Recompensa de centralidade: 3x
      • Recompensa de velocidade: 2x
      • Penalizao para curvas fechadas: 1x
      • Recompensa por progresso: 0.5x

Os fatores multiplicativos, expoentes e velocidades foram baseados nas implementaes das postagens analisadas. Esses parmetros iniciais podero ser ajustados futuramente com base no desempenho observado, buscando otimizar a funo de recompensa.

def reward_function(params):
    """
    Funo de recompensa para direo suave, rpida e consistente
    """
    # Extrair parmetros
    track_width = params['track_width']
    distance_from_center = params['distance_from_center']
    steering_angle = params['steering_angle']
    speed = params['speed']
    steps = params['steps']
    progress = params['progress']
    heading = params['heading']
    all_wheels_on_track = params['all_wheels_on_track']

    # Penalizao por sair da pista
    if not all_wheels_on_track:
        return 1e-3

    # Recompensa por adeso  linha central
    center_reward = max(1e-3, 1 - (distance_from_center / (0.5 * track_width))**0.4)

    # Penalizao para curvas muito fechadas e alta velocidade
    if abs(steering_angle) > 15 and speed > 1.8:
        steering_penalty = 0.5  # Reduz recompensa em curvas fechadas com alta velocidade
    else:
        steering_penalty = 1.0

    # Recompensa por velocidade ideal de acordo com o ngulo
    if abs(steering_angle) < 10:  # Retas
        speed_reward = max(1e-3, 1 - abs(speed - 2.5) / 2.5)

    else:  # Curvas
        speed_reward = max(1e-3, 1 - abs(speed - 1.8) / 1.8)

    # Recompensa por progresso (proporcional  eficincia dos passos)
    progress_reward = (progress / steps) * 10 if steps > 0 else 0

    # Combinao ponderada das recompensas
    reward = (center_reward * 3) + (speed_reward * 2) + (steering_penalty) + (progress_reward * 0.5)

    return float(reward)

Segunda Funo de Recompensa

A segunda funo de recompensa foi projetada com foco em promover consistncia no comportamento do veculo, incorporando informaes do estado anterior para suavizar transies e penalizar aes abruptas. Essa abordagem considera trs critrios principais: centralidade, consistncia na velocidade e suavidade nas curvas.

  1. Adeso Linha Central:

    • Assim como na primeira funo, o veculo recompensado de acordo com a proximidade ao centro da pista, penalizando desvios de forma progressiva.

  2. Consistncia na Velocidade:

    • A funo compara a velocidade atual com a anterior para penalizar mudanas bruscas. Isso incentiva o carro a manter um ritmo constante, mesmo em diferentes trechos da pista.

  3. Suavidade nas Curvas:

    • A funo avalia a mudana no ngulo de direo entre estados consecutivos. Mudanas bruscas no ngulo so penalizadas, incentivando curvas mais suaves.

  4. Penalidade para Aes Crticas:

    • Caso uma ao "imperdovel" ocorra, como sair da pista, uma penalidade severa aplicada, reduzindo a recompensa para o valor mnimo possvel.

  5. Combinao Ponderada:

    • Os critrios so combinados com pesos para equilibrar os objetivos:
      • Recompensa de centralidade: 2x
      • Recompensa de consistncia na velocidade: 3x
      • Recompensa de suavidade nas curvas: 2x

class PARAMS:
    """
    Classe para armazenar o estado anterior dos parmetros de conduo.
    """
    prev_speed = None  # Velocidade anterior do veculo
    prev_steering_angle = None  # ngulo de direo anterior
    prev_direction_diff = None  # Diferena de direo anterior (no usada no cdigo atual)
    prev_normalized_distance_from_route = None  # Distncia anterior normalizada ao centro da rota
    unpardonable_action = False  # Flag para aes crticas no perdoadas

def reward_function(params):

    # Extrai os parmetros fornecidos
    track_width = params['track_width']
    distance_from_center = params['distance_from_center']
    steering_angle = params['steering_angle']
    speed = params['speed']
    all_wheels_on_track = params['all_wheels_on_track']

    # Penaliza se o carro sair da pista
    if not all_wheels_on_track:
        return 1e-3

    # Calcula a recompensa baseada na proximidade do centro da pista
    center_reward = max(1e-3, 1 - (distance_from_center / (0.5 * track_width))**0.4)

    # Calcula a recompensa para consistncia na velocidade
    if PARAMS.prev_speed is not None:
        speed_diff = abs(speed - PARAMS.prev_speed)
        speed_reward = max(1e-3, 1 - speed_diff / 2)  # Penaliza variaes grandes de velocidade

    else:
        speed_reward = 1.0

    # Calcula a recompensa para suavidade nas curvas
    if PARAMS.prev_steering_angle is not None:
        steering_change = abs(steering_angle - PARAMS.prev_steering_angle)
        steering_reward = max(1e-3, 1 - steering_change / 15)  # Penaliza mudanas bruscas na direo
        
    else:
        steering_reward = 1.0

    # Combina as recompensas ponderadas
    reward = (center_reward * 2) + (speed_reward * 3) + (steering_reward * 2)

    # Aplica penalidade caso tenha ocorrido uma ao crtica no perdovel
    if PARAMS.unpardonable_action:
        reward = 1e-3

    # Atualiza o estado anterior com os valores atuais
    PARAMS.prev_speed = speed
    PARAMS.prev_steering_angle = steering_angle
    PARAMS.prev_normalized_distance_from_route = distance_from_center

    return float(reward)

def reward_function(params):
    """
    Funo de recompensa para direo suave, rpida e consistente
    """
    
    # Extrair parmetros
    track_width = params['track_width']   # largura da pista
    distance_from_center = params['distance_from_center'] # distncia do centro da pista
    steering_angle = params['steering_angle'] # ngulo de direo do carro
    speed = params['speed'] # velocidade do carro
    steps = params['steps'] # nmero de passos tomados (0:Nstep)
    progress = params['progress'] # progresso do carro ao longo da pista (0:100)
    all_wheels_on_track = params['all_wheels_on_track'] # se o carro est na pista

    min_reward = 1e-5
    expected_steps = 700 #baseado em treinamentos anteriores 

    # Penalizao por sair da pista
    if not all_wheels_on_track:
        return min_reward

    # Recompensa por adeso  linha central
    center_reward = max(min_reward, 1 - (distance_from_center / (0.5 * track_width))**0.4)

    # Recompensa por direo suave / rpida dependendo do angulo de direo

    if abs(steering_angle) > 15: #angulo maior que 15, velocidade maxima de 1.5
        speed_reward = max(min_reward, 1 - (abs(speed - 1.5) / 1.5))

    elif abs(steering_angle) > 5: #angulo maior que 5, velocidade maxima de 2.5
        speed_reward = max(min_reward, 1 - (abs(speed - 2.5) / 2.5))

    else: #angulos de at 5, velocidade maxima configurada
        speed_reward = 1

    # Recompensa por progresso (proporcional  eficincia dos passos)
    expected_steps_progress = expected_steps * (progress / 100) #numero de passos esperados para o progresso atual
    
    if expected_steps_progress > 0:
        progress_reward = max(min_reward,  (expected_steps_progress - steps)/expected_steps_progress  )**0.4
    else: progress_reward =0 

    # Combinao ponderada das recompensas (todos variam de 0 a 1)
    reward = (center_reward) + (speed_reward) + (progress_reward)

    return float(reward)
